/ mismeasured / operation-cleanup / reviews
displaytitle
  • Detects pages where the {{DISPLAYTITLE}} template is either used inappropriately or not used where it should be.
name_parameter
  • Articles about cosmetic items shouldn't use the 'name' parameter as it's added automatically.
wikipedia_template
  • Detects {{w}} templates that shouldn't be used anymore.
if_lang
  • The {{if lang}} template should only be used in templates as these are required to be dynamic.
no_label
  • Links that end with '/lang' should have a label to display a different text.
wrong_category
  • Detects pages that are in an English category. Some exceptions are present, often enough media files.
localized_template
  • Detects usages of templates that appear to be localized (ending in '/lang'). Instead, templates should offer translation switching.
no_loadout_name
  • Often enough, the 'loadout-name' parameter in the {{Item infobox}} template should be used.
external_links
  • Detects external links that don't have a {{lang icon|en}} attached to them, as almost always external links lead to English pages.
wrong_wikilinks
  • Detects wikilinks that are leading to a wrong language. Redirects are resolved to eliminate localized redirects.
wrong_wikipedia_links
  • Detects Wikipedia links that are leading to English language pages.
missing_wikipedia_pages
  • Detects Wikipedia links to pages that don't exist. Useful since red links aren't displayed in this case.
